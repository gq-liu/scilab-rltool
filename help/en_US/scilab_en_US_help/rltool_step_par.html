Step response parameters

Step response parameters

Description

The following step response parameters can be seen: rise time, settling time, peak time, percent peak overshoot, damping ratio (second order approximation). There are two ways to view these. (1) Select 'Response-->Details' and then 'Step'. This will bring up a menu that shows the numerical values of these parameters

(2) Select 'Response-->Closed loop'. Select ''Step response parameters'' in the menu that comes up.
