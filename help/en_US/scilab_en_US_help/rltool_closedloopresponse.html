Closed loop dynamic response

Closed loop dynamic response

Description

The following closed loop dynamic responses can be seen: Impulse, Step, Ramp, (arbitrary) piece-wise linear single-valued-input, sinusoidal input.

Your current gain (as shown on the slider) is used to compute these responses. The closed loop transfer function can be computed either with gain in the forward path or the feedback path

A selection menu is displayed when you select ''Response-->Closed loop''. Click on the required dynamic response from this menu.

The maximum simulation time und step interval can be selected in 'Settings->Dynamic Response'.
