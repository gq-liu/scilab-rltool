Sensitivity plot

Description

Sensitivity plot can be seen by selecting 'Response->Closed loop'. This brings up a menu with all closed loop responses that Rltool can display. Select 'Sensitivity plot' to view the Sensitivity plot.

The default minimum frequency is 0.01 rad/sec. The default maximum frequency is 1000 rad/sec. These can be changed by changing 'Settings->Sensitivity Plot'.

The Sensitivity plot can be displayed in Hz. as well as rad/sec. by setting the appropriate frequency units in 'Settings->Frequency units'.
