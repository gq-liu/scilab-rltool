Nyquist and Nichols plots

Plot of (open loop) transfer function

Description

Select 'Response->Nyquist' for Nyquist Plot of (open loop) transfer function. The default minimum frequency is 0.01 rad/sec. The default maximum frequency is 1000 rad/sec. To change these: 'Settings->Nyquist-Nichols Plot'.

Select 'Response->Nichols' for Blacks(Nichols) Plot. Default values for Nichols and Nyquist plots are the same. You can see these plots both in rad/sec and Hz units. using the 'Settings-->Frequency units' option.
